# Dataset Schema

| Column Name | Description | Unit | Type |
|------------|-------------|------|------|
| date | Observation date | YYYY-MM-DD | Feature |
| city | City name | Text | Feature |
| latitude | Latitude | Degree | Feature |
| longitude | Longitude | Degree | Feature |
| T2M | Temperature at 2m | °C | Feature |
| RH2M | Relative Humidity | % | Feature |
| WS10M | Wind Speed at 10m | m/s | Feature |
| PS | Surface Pressure | kPa | Feature |
| CLOUD_AMT | Cloud Amount | Fraction (0-1) | Feature |
| PRECTOTCORR | Rainfall | mm/day | Feature |
| ALLSKY_SFC_SW_DWN | Solar Radiation (GHI) | kWh/m²/day | Target |
